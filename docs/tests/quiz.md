## 出题流程

### TC-QUIZ-001: 正常生成题目

**优先级**：P0
**关联**：US-002, T721
**前置条件**：
- 用户已登录
- 已有知识点"心力衰竭的 NYHA 分级"
- AI 配置正确（API Key 有效）

**步骤**：
1. 进入 `/quiz` 页面
2. 选择知识点"心力衰竭的 NYHA 分级"
3. 设置题目数量为 3
4. 点击"生成题目"

**预期结果**：
- 显示加载状态
- 生成 3 道 A1/A2 型选择题
- 每道题包含：题干、4 个选项（A/B/C/D）、正确答案、解析
- 题目内容与知识点相关

**状态**：🔲 待测试

## API 层

### TC-QUIZ-020: POST /api/quiz/generate 成功

**优先级**：P0
**关联**：T721
**前置条件**：
- 有效 Cookie + AI 配置正确

**步骤**：
1. 发送 `POST /api/quiz/generate`，body:
   ```json
   {
     "knowledgePointIds": ["kp_001"],
     "count": 3
   }
   ```

**预期结果**：
- 返回 200
- body 包含 `questions` 数组（3 个元素）
- 每个 question 包含 `stem`, `options`, `correctAnswer`, `explanation`

**状态**：🔲 待测试
